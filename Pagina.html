Módulo 02: Hyper Text Markup Language - HTML


HTML é uma linguagem de marcação de hypertexto. O HTML usa "Marcação" para anotar texto, imagem e outros conteúdos para exibição em um navegador da Web.


 O HTML é o tipo de código usado para estruturar páginas web e seu conteúdo.Essa linguagem trabalha em concomitância com os navegadores de internet: Google Chrome, Firefox, Opera e outros. Cabe ao navegador interpretar e renderizar o site de acordo com as instruções contidas no código HTML da página. Para isso, a linguagem de marcação utiliza alguns comandos que identificam diferentes estruturas dentro do site em HTML, são elas:

  1. Tags: são escritas entre símbolos “<” e “>”, serve para indicar a função do elemento. As tags precisam ser fechadas para indicar o fim do conteúdo, para isso utiliza-se o par de menor e maior que com a “/” antes da tag no fechamento.
  2. Atributos: são utilizados dentro da tag de abertura para controlar o comportamento do elemento.

Existe uma gama de tags na linguagem HTML, entre as quais:

  • h1, h2, h3, h4, h5: adiciona títulos a página, cujos tamanhos variam de h1 à h6 em ordem decrescente;
  • a: insere links na página através do uso do atributo href;
  • img: serve para inserir imagens na página, requer o uso do atributo scr para determinar a fonte da figura.
  • p: cria um novo paragráfo no texto;
  • b: de bold, serve para deixar o elemento em negrito;
  • br: adiciona quebra de linha;
  • button: insere botões em uma página;
  • ol: comando para formação de listas ordenadas;
  • ul: comando para listas desordenadas;

Essas são só algumas das tags existentes na linguagem de marcação descrita que são utilizadas para a criação de uma página HTML que são utilizadas para o desenvolvimento Frontend em conjunto com o CSS, para formação da base das páginas web.
